## 흐름제어 if, for while

#### Algorithms : 1~100 사이 숫자 맞추기

```python
import random
n = random.randint(1, 100)
while True :
    guess = int(input("정답을 맞춰보세요. :"))
    if n == guess :
        print("정답입니다.")
        break
    elif abs(n - guess) < 10 :
        print("아 아깝습니다.")
    else : 
        print("틀렸습니다.")
```

#### fizz buzz
1) 1 ~ 100까지 숫자를 순서대로 출력한다
2) 그 숫자가 3의 배수일 때는 숫자 대신 fizz
3) 그 숫자가 5의 배수일 때는 숫자 대신 buzz
4) 그 숫자가 15의 배수일 때는 숫자 대신 fizzbuzz 를 출력

```python
for i in range(1, 101):
    if i % 15 == 0 :
        print('fizz')
    elif i % 5 == 0 : 
        print('buzz')
    elif i % 3 == 0 :
        print('fizzbuzz')
    else : 
        print(i)
```

#### Algorithms : 소수 판별

1) 1 ~ 1000까지 숫자 중에 소수만 출력

소수는 1과 자기자신으로만 나누어 떨어지는 숫자
```python
for i in range(1, 1001):
    key = True
    for j in range(2, i):
        if i % j == 0:
            key = False
            break
    if key:
        print(i)
```
